## Tools

### [deploy.sh](deploy.sh): Deploy the folder to the $SKY_PATH_BIN/skz folder

```
Usage: deploy <folder> <name> [all]

example:
    deploy path/to/turbopixel turbopixel
```

### [skz.sh](skz.sh): Create a .skz distributable archive

```
Usage: skz <folder> <name> <output (skz)>

example:
    skz path/to/turbopixel turbopixel turbopixel-1.0.0.skz
```
